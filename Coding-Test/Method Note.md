### length VS length()

배열의 크기를 구할 때는 `length`

문자열의 길이를 구할 때는 `length()`

### String to int

```java
String s = "0817";
int i = Integer.parseInt(s);
```

### int to String
```java
int i = 16;
String str = String.valueOf(i);
```

### String to char []
```java
String s = "abcdefg";
char[] c = s.toCharArray();
```
