### `@RestController`

1.  요청 ... 어떤 지정  
    select -> @GetMapping  
    delet -> @DeleteMapping  
    insert -> @PostMapping  
    update -> @PutMapping

#### <b>GET</b> 요청에서는 추가되는 객체를 받을 때, `@RequestParam`를 사용한다.

#### <b>POST</b> 요청에서는 추가되는 객체를 받을 때, `@RequestBody`를 사용한다.

{name : value} 보낼 때는 Json 형식 || [Object object] 받을 때 형식
